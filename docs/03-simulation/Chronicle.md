# Purpose

Defines the Chronicle system — one of the game's flagship systems. The Chronicle automatically records major historical events, transforming each save into a unique historical document that captures the story of the player's kingdom.

---

# Overview

Rather than merely tracking objectives, the game records history. The Chronicle is an automated historical log that captures significant events as they occur in the simulation. Each kingdom develops a unique history, making every save file a personal historical document.

The Chronicle is one of the game's defining systems and directly supports the vision of emergent stories and long-term attachment.

---

# Design Goals

1. **Automatic recording.** Events are captured without player input — the Chronicle writes itself.
2. **Emergent history.** The best stories should come from simulation rather than scripted events.
3. **Unique per kingdom.** No two Chronicle entries should be identical because no two kingdoms evolve identically.
4. **Six-Month Return Test.** The Chronicle should make returning players smile when they scroll through their kingdom's history.
5. **Emotional attachment.** The Chronicle transforms gameplay events into a personal history worth revisiting.

---

# Non-Goals

- The Chronicle is not a quest log or objective tracker.
- The Chronicle is not a player-authored journal (entries are generated by the simulation).
- The Chronicle is not a performance or achievement scoreboard.

---

# Event Types

The source material specifies the following event types that the Chronicle records:

| Event Type | Description |
|------------|-------------|
| Births | A villager or hero child is born. |
| Marriages | Two villagers or heroes marry. |
| Deaths | A villager or hero dies. |
| Colony Founding | A new colony is established. |
| Discoveries | A new region, resource, or technology is discovered. |
| Disasters | Fires, revolts, or other negative events occur. |
| Hero Achievements | A hero accomplishes something notable. |
| Completed Wonders | A major building or wonder is completed. |
| Wars | Conflicts begin or end. |
| Technological Advances | A research milestone is reached. |

---

# Event Structure

Each Chronicle entry should capture, at minimum:

| Field | Description |
|-------|-------------|
| Date | In-game date/timestamp of the event. |
| Type | One of the event types listed above. |
| Description | Narrative text generated from the event data. |
| Participants | Villagers, heroes, or settlements involved. |
| Location | Where the event occurred (capital, colony, world map). |

The exact format and generation mechanism for narrative descriptions is not specified in the source material.
