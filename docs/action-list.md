# Action Items

## Research: Free & Open-Source Game Assets

**Status:** Pending

Look for free, open source game assets that could be used in The Long Reign — art, audio, fonts, tilesets. Evaluate licensing compatibility (MIT, GPL, CC0, CC-BY, etc.) against the project's license (TBD but will be open source).

Key sources to check:
- [OpenGameArt.org](https://opengameart.org) — CC0 / CC-BY / GPL art, tilesets, sprites
- [Kenney.nl](https://kenney.nl) — CC0 / public domain game assets
- [Freesound.org](https://freesound.org) — sound effects / ambient audio (CC0/CC-BY)
- [itch.io](https://itch.io) — free asset packs (check individual licenses)
- [Sprite Database](https://spritedatabase.net) — retro sprite collections (verify license per entry)
- [Sonniss Game Audio bundles](https://sonniss.com/gameaudiogdc) — large CC0 sound effect libraries

Considerations:
- Pixel art style compatibility (see `docs/07-art/Pixel Art Style.md`)
- Audio direction alignment (see `docs/07-art/Audio.md`)
- Whether custom assets are preferable for visual consistency
- Document findings and recommendations in `docs/07-art/` or a new sub-page
